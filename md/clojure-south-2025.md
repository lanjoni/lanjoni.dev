# Building elegant web applications with ClojureScript, React, and UIx! (Clojure South 2025)

[Clojure South is back in 2025](https://clojure-south.com), with the second edition scheduled to discuss Clojure, Functional Programming, and more at the biggest Clojure event in Latin America!

In this new edition, I had the pleasure of presenting and discussing UIx, a new ClojureScript interface for React that leverages the best of both worlds: it's tiny and brings React's style of usage closer to how we code Clojure nowadays.

This post is just publish some useful links of the presentation, and also add more information about it! I hope you enjoy writing some parentheses to create cool web interfaces.
